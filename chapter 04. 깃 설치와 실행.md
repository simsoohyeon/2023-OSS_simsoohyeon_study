## 1. 깃 설치와 편집기 설정
```
Visual Studio code로 설정
- 새 저장소의 기본 브랜치 이름 설정
: $ git config --global init.defaultBranch main
- 깃 설치 폴더 확인
: C:\Program Files\git
```

## 2. 깃 실행
- CLI 인터페이스: 명령 프롬포트와 같은 인터페이스
```
- $ pwd: 현재 작업중인 경로
- $ ls-al: 디렉토리에 있는 파일 목록 + 세부 정보
  (프롬포트 첫 줄: 현재 디렉토리 표시)
- $ git --version: 버전 확인
- $ git config --list: 설정 확인
- $ git --help: 도움
```
